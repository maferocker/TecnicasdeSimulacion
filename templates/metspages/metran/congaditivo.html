Método congruencial lineal

Este algoritmo requiere una secuencia previa de \(n\) números enteros \(x1, x2, x3...xn\) para generar una nueva secuencia de números enteros que empiezan en \(xn+1, xn+2, xn+3...\). Su ecuación recursiva es: \[ Xi = (Xi-1, + Xi-n)mod(m) \ i=n+1,n+2,n+3,....,N \] {% if N != None %} Teniendo los datos iniciales se procede a multiplicar la semilla con el multiplicador, para luego sumarle el incremento: \[ ({{ X0 }} * {{ A }}) + {{ C }} = {{ res1 }} \] A este se le aplica el módulo que designamos, el cual nos va a dejar solo el residuo de la división realizada por este, con esto habremos descubierto cual es nuestro valor pseudorandómico y nuestra nueva semilla para la siguiente iteración: \[ { {{ res1 }} \over {{ M }} }= {{ res2 }} \qquad {{ res2 }}* {{ M }} = {{ res1 }}\] Esto se puede apreciar de mejor manera en la siguiente iteración como se ve a continuación usando la nueva semilla: \[ ({{ res1 }} * {{ A }}) = {{ res3 }} \qquad { {{ res3 }} \over {{ M }} }= {{ res5 }} = {{ res4 }} \] De ahí solo hay que repetir el proceso las veces necesarias teniendo en cuenta el reemplazar la semilla en cada iteracion. {% endif %}
